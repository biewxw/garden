## Potência

### Introdução

Dentro da matemática, possuimos o conceito de potência.Esse conceito matemático nada mais é do que uma forma de escrever uma multiplicação (produto) de fatores iguais. Para exemplificar de uma forma mais clara podemos observar os exemplos abaixo:

_a) 2.2.2.2 = 2^4_</br>
_b) (-3).(-3).(-3).(-3) = (-3)^5_

### Análise

**De forma geral podemos observar:**</br>
_(a^n = a.a.a...a)_ onde _a_ é o produto e _n_ a quatidade de vezes que o produto vai ser multiplicado pelo seu semelhante)

### Casos especiais

**Elevação com números negativos**</br>
Para resolver casos onde possuímos uma elevação utilizando um número negativo, devemos apenas inverter a fração. Feito isso removemos o negativo do expoente. Por exemplo:

_i) a^-1 = (a/1)^1 = 1/a_</br>

**Elevação ao expoente 0**</br>
Podemos encontrar em alguns casos a elevação sendo feita utilizando o expoente 0. Nesses casos, sempre que o expoente for igual a 0, o valor final do produto será igual a 1. Por exemplo:

_i) a^0 = 1_</br>

**Expoente fracionado**</br>
Em casos onde o expoente de um produto é fracionado, podemos resolver tal caso transformando o cálculo em raiz quadrada. Por exemplo:

_i) a^n/n = raiz(a^n, indice(n))_</br>
